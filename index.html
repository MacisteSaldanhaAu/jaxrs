¿Qué es REST?

REST o Representational State Transfer es una arquitectura para el desarrollo de Web Services que se ejecuta sobre HTTP. Utiliza los métodos del protocolo para realizar operaciones sobre los recurdos disponibles.

Métodos HTTP vs Operaciones REST.
HTTP Verb CRUD
POST Create
GET Read
PUT Update / Replace
PATCH Update / Modified
DELETE Delete

¿Qué es JAX-RS?

Java API for RESTful Web Services o JAX-RS, es la especificación de Java que provee soporte para crear web services REST.

Configuración

Para configurar un proyecto que exponga servicios REST utilizaremos: eclipse juno, maven, jboss & zk framework.

Realizar los siguientes pasos en eclipse:

  • 1 - Crear un nuevo projecto maven : File --> New --> Project
  • 2 - Seleccionar zk-archetype-webapp
  • 3 - Especificar el Group Id, Artifact Id & package de la aplicación
  • 4 - Finalizar el wizard

Para permitir el soporte de JAX-RS en la aplicación para con el servidor jboss modificar el archivo pom.xml agregando la siguiente dependencia.

Se requiere agregar una clase que extienda de javax.ws.rs.core.Application para de esta forma indicar al servidor de aplicaciones que se requiere el despliegue de Web Services REST.

package com.hercarr.jaxrs.sample.ws.rest;

import javax.ws.rs.ApplicationPath;
import javax.ws.rs.core.Application;

@ApplicationPath("/rest")
public class RestfulApplicationActivator extends Application {

}

Posteriormete, agregue los recursos necesarios así como las operaciones requeridas para cada uno de ellos..

package com.hercarr.jaxrs.sample.ws.rest;

import java.util.List;

import javax.ws.rs.GET;
import javax.ws.rs.PathParam;

@Path("/users")
public class UserResource {
        
    @GET
    @Path("/{name}")
    public String helloWorld(@PathParam("name") String name) {
        return "Hi " + name + ", Hello World!!!";
    }

}

Resultado

Para ver el resultado del Web Service anterior en su browser introduzca la siguiente URL http://localhost:8080/JAXRSSample/rest/users/nombre sustituyendo el valor nombre por su nombre.

El navegador debería mostrar un mensaje como el siguiente.

Hi hercarr, Hello World!!!
